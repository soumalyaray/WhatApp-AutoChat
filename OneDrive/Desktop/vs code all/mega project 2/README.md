# WhatsApp AutoChat

Automated WhatsApp interaction using pyautogui and pyperclip for mouse-based text selection, copying, and Chrome automation.

## Files
- `main.py`: Logs mouse positions for automation setup.
- `program.py`: Main script - clicks Chrome, selects/drags text, copies to clipboard.

## Requirements
```
pip install pyautogui pyperclip
```

## Usage
Run `python program.py` (coordinates may need adjustment for your screen).

**Note:** Use responsibly; respects WhatsApp ToS.
